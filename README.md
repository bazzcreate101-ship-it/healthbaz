# Healthbazz

Platform e-commerce untuk apotek dan pengiriman obat, terinspirasi dari Halodoc.

## Memulai

Instal dependensi:
```bash
npm install
```

Jalankan server pengembangan:
```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) dengan browser Anda untuk melihat hasilnya.

## Fitur

- Desain modern dan responsif
- Katalog obat dan pencarian
- Antarmuka ramah pengguna
- Performa cepat dan dioptimalkan

## Teknologi

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- ESLint
